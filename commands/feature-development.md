# Feature Development

Use the `superpowers:feature-development` skill to run the complete feature development workflow.

**Use the Skill tool now:** `Skill(skill="feature-development")`

This skill orchestrates 5 phases:
1. Setup (git worktree)
2. Design + Planning (brainstorming)
3. Implementation (executing-plans)
4. Manual Testing
5. Completion (finishing-branch)
